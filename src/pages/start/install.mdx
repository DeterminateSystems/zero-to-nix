First, download our Nix installer <Snippet id="harmonic" show="Harmonic" client:only="react" />.
This is not yet the official installer for Nix/ NixOS, however this will probably change in the future.

```shell
$ curl https://harmonic.dev | sh
```

Harmonic follows different steps based on the *build plan* you select.

Currently there are only two: `linux-multi` (for 64-bit Linux systems) and `darwin-multi` (for 64-bit Intel and ARM macOS systems).
The `-multi` refers to the type of installation being performed (meaning [‚Äúmulti-user‚Äù]).

If this seems confusing, don‚Äôt worry about it too much for now.

Once Harmonic has finished you should verify the installation by running:

```bash
$ nix --help
```

This should print the help output of the *nix* command.
